# ppt-skill

`ppt-skill` 是一个用来生成 PPT 的通用技能。

你可以把它理解成一个“PPT 制作助手”：用户给一个主题、一份大纲，或者一份公司 PPT 模板，它会把内容整理成一份 `.pptx` 文件。

它不只适合产品提案，也可以用于项目汇报、培训课件、路演材料、活动方案、公司介绍、竞品分析、工作总结等场景。

## 它能做什么

- 根据用户提供的 PPT 大纲制作完整 PPT。
- 用户只说一句主题时，先帮用户整理一份合适的大纲，再制作 PPT。
- 用户提供公司或品牌 PPT 模板时，沿用模板的视觉风格，把新内容填进去。
- 用户没有模板时，使用内置主题生成 PPT。
- 没有图片也可以生成；有 logo、截图、产品图时，也可以放进 PPT。
- 支持封面、痛点、方案、能力、流程、路线图、表格、数据、下一步、公司介绍等常见页面。

## 适合哪些场景

你可以用它来做：

- 产品提案
- 客户方案
- 项目汇报
- 培训课件
- 路演材料
- 活动策划
- 公司介绍
- 竞品分析
- 年度总结
- 商业复盘

## 使用时会怎么判断

这个技能会优先看用户有没有提供 PPT 模板和大纲。

如果用户上传了 PPT 模板，就直接使用这个模板。模板决定 PPT 长什么样，比如颜色、背景、版式和品牌风格。

如果用户提供了 PPT 大纲，就按这个大纲来做。大纲决定讲什么、讲几页、按什么顺序讲。

如果用户只给了一句话主题，比如“帮我做一份 AI 客服降本增效方案 PPT”，那就先根据主题、用途和受众生成一份建议大纲。页数不是固定的，简单内容可以短一些，复杂方案可以长一些。

可以简单理解为：

```text
PPT 模板：决定长什么样
PPT 大纲：决定讲什么
PPT 生成器：把内容做成 .pptx 文件
```

## 关于图片

这个技能本身不负责生成图片。

如果没有图片，它会用文字、色块、卡片、表格、流程和数字排版来完成 PPT。

如果用户提供了 logo、产品图、截图、流程图、团队照片等图片，可以把这些图片放进 PPT。

如果其他工具已经生成了图片，也可以把图片文件交给这个技能使用。

## 内置主题

用户没有提供模板时，可以使用内置主题。

| 主题 | 适合场景 |
|---|---|
| `default` | 通用商务汇报、内部简报 |
| `blue` | 科技、AI、数据、平台类主题 |
| `green` | 健康、环保、农业、公益类主题 |
| `orange` | 销售方案、营销活动、增长方案 |
| `purple` | 品牌、设计、咨询、高端服务 |
| `red` | 政务、年会、正式活动 |
| `teal` | 医疗、教育、服务设计 |

## 常见使用方式

### 1. 用户提供大纲

用户可以直接给出类似这样的内容：

```text
帮我生成 PPT，以下为 PPT 大纲：

1. 封面：玻尿酸面膜与普通面膜有什么不同
2. 痛点导入：为什么普通补水面膜效果短暂
3. 概念定义：什么是普通面膜
4. 概念定义：什么是玻尿酸面膜
5. 核心差异对比
...
```

技能会按用户的大纲制作 PPT，不会强行改成固定页数。

### 2. 用户只提供主题

用户也可以只说：

```text
帮我做一份 AI 客服降本增效方案 PPT
```

这时会先生成一份建议大纲。默认会做成标准汇报长度，但会根据主题复杂度调整页数。

### 3. 用户提供模板

如果用户提供了公司 PPT 模板，就会优先使用模板：

```text
请用这个公司模板，帮我做一份项目汇报 PPT。
```

模板负责视觉风格，大纲或主题负责内容。

## 命令行用法

如果你是在本地或其他 Agent 环境中使用，可以通过命令行生成 PPT。

安装依赖：

```bash
uv pip install python-pptx
```

查看可用主题：

```bash
ppt-skill list-themes
```

使用内置主题生成：

```bash
ppt-skill create \
  --theme purple \
  --slides slides.json \
  --images '{}' \
  --output output.pptx
```

使用 PPT 模板生成：

```bash
ppt-skill create \
  --template brand-template.pptx \
  --slides slides.json \
  --images '{}' \
  --output output.pptx
```

如果当前环境没有安装 `ppt-skill` 命令，也可以直接运行脚本：

```bash
python scripts/ppt_skill/cli.py create --theme blue --slides slides.json --images '{}' --output output.pptx
```

## 内容格式

实际生成 PPT 时，内容会被整理成一组页面数据。普通用户不需要手写这些数据，通常由 Agent 自动整理。

一个简化示例：

```json
[
  {
    "type": "cover",
    "title": "玻尿酸面膜与普通面膜有什么不同",
    "subtitle": "洞察差异 · 抢占功效护肤新风口"
  },
  {
    "type": "problem",
    "title": "痛点导入",
    "items": [
      "补水面膜天天敷，皮肤依然干燥",
      "敷完水润一时，第二天打回原形",
      "传统面膜更多是表面滋润，而非深层保湿"
    ]
  }
]
```

支持的页面包括封面、背景痛点、方案概述、能力介绍、流程步骤、路线图、预算表、价格表、价值数据、下一步计划、保障体系、交付计划、公司介绍等。

更详细的格式见 [SKILL.md](SKILL.md) 和 [references/page_types.md](references/page_types.md)。

## 使用建议

- 如果有公司模板，优先提供模板。
- 如果已经有大纲，直接提供大纲。
- 如果只有主题，也可以直接说主题，让技能先生成大纲。
- 如果有关键数据，请直接提供；不要让工具猜测真实数据。
- 如果内容很多，建议拆成多页，不要挤在一页里。
- 如果需要放图片，请提供本地图片文件路径。
